Installation
============

Precondition: Raspian Linux (http://www.raspberrypi.org/downloads) + libcurl4-gnutls-dev + libconfig-dev

Compile: sudo gcc -o /usr/sbin/1wirevz /tmp/1wirevz.c -lconfig -lcurl

1wirevz.cfg		-> /etc/

modules  		-> /etc/

rc.local  		-> /etc/

1wirevz 		-> /etc/init.d/
